판단 기준:
- 저장 O: 개념/원리/구조/동작 방식을 묻고 설명한 경우
- 저장 X: 단순 작업 요청(파일 만들어줘, 커밋해줘, 푸시해줘 등)이거나 잡담

저장할 경우 절차:
1. docs/memo/ 하위 파일들을 Read로 먼저 확인한다
2. 같은 주제 파일이 있으면 내용 검토 후 보강할 내용이 있으면 append, 이미 완전하면 skip
3. 새 주제면 영어 kebab-case.md 로 새 파일 생성
4. Q&A 형식(## Q / ## A)으로 정리하되 관련 개념을 보강해 완전한 학습 문서로 만들 것
5. 한국어로 작성

저장하지 않을 경우 아무것도 하지 말고 응답도 출력하지 말 것.
